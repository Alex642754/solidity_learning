## 📘 项目简介

本仓库用于记录我在学习 **Solidity 智能合约** 与 **Hardhat 框架** 过程中的代码与心得。  
内容包括智能合约的语法理解、EVM 执行逻辑、合约部署与交互、测试脚本编写等。  
每一个学习节点都会在 `contracts/` 目录中以 **DayX_主题.sol** 的形式保存对应的合约文件。
